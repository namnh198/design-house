# How to install

```
composer install
cp .env.example .env
php artisan key:generate
php artisan serve
npm install
npm run dev
```

## Configurate URL
```
APP_URL=http://localhost:8000
CLIENT_URL=http://localhost:3000
```

## Configurate Database
```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=designhouse
DB_USERNAME=root
DB_PASSWORD=hoainam
```
